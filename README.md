# cyphers-Discord-Bot

## init Bot

using node-js 14
install from https://nodejs.org/

1. change API Tokens
   change \`config.ts\`

    > CyphersApiKey -https://developers.neople.co.kr/contents/apiDocs/cyphers
    >
    > DiscordApiKey -https://discord.com/developers/applications

2. run below commands

```
yarn install
yarn build
yarn typeorm schema:sync
```

## run cyphers bot

```
yarn start
```
